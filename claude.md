# Project: Kotlin AI IDE

## Objectif
Création d'un IDE Kotlin avec assistant IA intégré, basé sur Theia et CodeLlama.

## Architecture
- Frontend: Theia + React
- Backend: Node.js
- AI: CodeLlama
- IDE: Eclipse Theia + Kotlin support
